Тема проекта:
Сервис по прокату автомобилей

    Веб-приложение, где пользователь может зарегистрироваться, выбрать подходящий автомобиль и забронировать его на нужные даты. Пользователь получает доступ к каталогу автомобилей с подробной информацией (модель, характеристики, стоимость аренды), а также возможность загрузки необходимых документов для оформления аренды. Сотрудник проката получает доступ к системе для проверки документов клиента и автоматической генерации пакета документов для выдачи автомобиля.

Цели:

    Возможность удалённого бронирования автомобилей получение информации о доступных автомобилях и их характеристиках выбор дат аренды и оформление заявки через сайт

    Автоматизация документооборота автоматическая генерация договора аренды, акта приёмки-передачи и доверенности на авто

    Упрощение процесса выдачи автомобиля минимизация времени на оформление документов при получении авто

Задачи:

    Реализовать каталог автомобилей с фильтром и поиском отображение свободных автомобилей на выбранные даты

    Создать систему загрузки и проверки документов клиента паспорт, водительское удостоверение, дополнительные документы
 
    Реализовать автоматическую генерацию документов договор аренды, акт приёмки-передачи, доверенность на автомобиль

    Реализовать личный кабинет для сотрудника проката управление бронированиями, подтверждение выдачи, возврат автомобиля
