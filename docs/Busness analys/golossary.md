Словарь терминов

1. Общие понятия
 
    Классический прокат (Rental): Долгосрочная аренда автомобиля (от суток и более) с фиксированной стоимостью за сутки и залоговым депозитом.

    Сервис / Платформа: Веб-сайт или мобильное приложение, предоставляющее пользователю доступ к функционалу аренды.

    Аккаунт / Личный кабинет: Персональный раздел пользователя в системе, содержащий его данные, историю аренд и настройки.

    Верификация: Процесс проверки предоставленных пользователем данных на подлинность и актуальность.

2. Пользователи и роли (Акторы)

    Гость (Unauthorized User / Guest): Неавторизованный посетитель сервиса, имеющий доступ только к просмотру информации.

    Клиент / Арендатор (Customer / Renter): Зарегистрированный пользователь, прошедший верификацию и имеющий право бронировать автомобили.

    Водитель (Driver): Лицо, непосредственно управляющее автомобилем.

    Администратор (Administrator / Admin): Сотрудник компании, управляющий работой сервиса через административную панель.

    Менеджер (Manager): Частный случай администратора, чаще отвечающий за взаимодействие с клиентами и оформление договоров.

3. Автомобили и автопарк
    Автопарк (Fleet): Совокупность всех автомобилей, доступных для аренды в сервисе.

    Транспортное средство (ТС) / Автомобиль (Car / Vehicle): Единица автопарка, объект аренды.

    Статус автомобиля (Car Status): Текущее состояние машины в системе.

    Характеристики автомобиля (Car Specs): Набор параметров, описывающих машину.

    Тариф (Tariff / Rate): Правила и стоимость аренды. Может зависеть от времени (час/сутки), пробега, класса автомобиля или акционных предложений.

4. Бизнес-процессы и заказы

    Бронь / Бронирование (Booking / Reservation): Предварительный заказ автомобиля на определенную дату и время, при котором машина закрепляется за клиентом, но аренда еще не началась.

    Аренда / Поездка (Rent / Trip): Период времени, в течение которого клиент фактически использует автомобиль.

    Старт аренды (Rent Start): Момент, с которого начинается отсчет времени аренды.

    Окончание аренды (Rent End): Момент завершения использования автомобиля, фиксирующий итоговую стоимость.

    Продление аренды (Extension): Увеличение времени аренды уже после ее начала.

    Итоговая стоимость (Total Cost): Окончательная сумма к списанию, рассчитанная по факту завершения аренды с учетом времени, пробега и дополнительных услуг.

6. Интерфейс и функциональность

    Каталог (Catalog): Раздел сервиса со списком всех доступных автомобилей и возможностью их фильтрации.

    Фильтры (Filters): Инструменты для поиска авто по параметрам.

    Административная панель (Admin Panel / Backoffice): Закрытая часть сервиса для сотрудников компании, предназначенная для управления системой.

8. Документы

    Договор аренды: Юридический документ, регламентирующий условия передачи автомобиля клиенту.

    Акт приёмки-передачи: Документ, фиксирующий состояние автомобиля, показания одометра и уровень топлива.

    Доверенность на ТС: Документ, предоставляющий клиенту право управления транспортным средством.

    Шаблон документа: HTML-файл с переменными для автоматической генерации PDF.
